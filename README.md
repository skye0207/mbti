# 人格译站

一键中译中，结合 MBTI 沟通风格，把上头情绪翻成好好沟通。

## 技术栈

- React
- Vite
- JavaScript
- Less
- localStorage

不使用 TypeScript，不使用 Tailwind CSS，不使用 UI 组件库。

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```

## 功能

- 首页 / 今日人格天气
- 消息翻译器
- MBTI 相处指南
- 冲突复盘
- 我的档案，本地保存 MBTI 和沟通偏好

## 设计参考

设计图放在：`docs/design-reference.png`。

本项目参考其粉紫渐变、圆角卡片、轻生活方式 App 的视觉气质，不做像素级复刻。
